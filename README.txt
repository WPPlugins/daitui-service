== Description ==

[代推](http://www.daitui.com/ "代推，收藏与分享书签按钮服务")，是一个主要针对中文用户而开发的收藏与分享按钮服务。

== Installation ==

1. 将插件目录`daitui-service/`上传至`/wp-content/plugins/`目录
2. 在后台插件管理页面激活插件
